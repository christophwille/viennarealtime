1) Download the .xsd files (starting point is http://www.wienerlinien.at/ogd_routing/XML_TRIP_REQUEST2?language=en)
2) In Notepad++ open all .xsd files and "Convert to UTF8" / "Save"
3) Run

xsd itd.xsd /c /n:WienerLinien.Api.Routing.Messages

4) Replace (in two separate runs)

[System.SerializableAttribute()]
[System.ComponentModel.DesignerCategoryAttribute("code")]

with blank (those attributes are not supported in this PCL)
